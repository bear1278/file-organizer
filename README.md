# File organizer

## Описание проекта
Программа организует файлы в структуру папок согласно их расширений. Программа сортирует файлы по папкам Музыка, Документы, Фото, Видео и Архивы. Если файл имеет расширение, которого нет в правилах сортировки, то файл остается в корне выбранной директории.

## Возможности

- Автоматическая сортировка файлов по расширениям
- Логирование всех операций
- Отчёт со статистикой
- Обработка конфликтов имён

## Установка

git clone https://github.com/bear1278/file-organizer.git
cd file-organizer
go build

## Использование

./file-organizer
